\subsection{Detailed Description}
\mbox{\hyperlink{class_sculptor}{Sculptor}} uma classe que projeta um conjunto de Voxels organizando-\/os para \char`\"{}esulpir\char`\"{} uma imagem. a classe pode esculpir caixas, esferas e elipsses a partir de voxels representados como cubos. bem como escrever as estruturas em arquivos \textquotesingle{}.off\textquotesingle{} ou \textquotesingle{}.vect\textquotesingle{}.
