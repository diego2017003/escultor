Detailed Description

Sculptor uma classe que projeta um conjunto de Voxels organizando-os para "esulpir" uma imagem. a classe pode esculpir caixas, esferas e elipsses a partir de voxels representados como cubos. bem como escrever as estruturas em arquivos '.off' ou '.vect'.
